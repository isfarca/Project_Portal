# Project_Portal

A teleport mechanic in Unity from the 'Portal'-Game.

## Control

'W, A, S, D' for the movement of the player.
'MOUSE' for turning the player.
'Left mouse button' to set blue portal.
'Right mouse button' to set red portal.
